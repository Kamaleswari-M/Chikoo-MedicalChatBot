## Chikoo-MedicalChatBot
Chikoo is an AI-powered medical assistant chatbot designed to provide general health information and symptom guidance through a user-friendly conversational
interface. The chatbot leverages Large Language Models (LLMs) to answer health-related queries, offer wellness tips, and provide educational information while 
encouraging users to seek professional medical advice when necessary.

The application includes voice input functionality, emergency symptom detection, chat history management, dark mode support, and an interactive chat interface. 
Chikoo aims to make healthcare information more accessible through an engaging AI-powered experience.

## Installation

1. Clone the repository
```bash
git clone https://github.com/Kamaleswari-M/Chikoo-AI-Medical-Assistant.git

cd Chikoo-AI-Medical-Assistant
```

2. Install dependencies
```bash
npm install
```

3. Create GROQ API KEY

Move to .env file inside the backend folder:

GROQ_API_KEY=your_groq_api_key

Replace with your own_groq_api_key

4. Start the Backend Server

Open terminal:

```bash
cd backend

node server.js
```

You should see:

🐶 Chikoo AI running on port 3000

5. Run the Frontend

Using VS Code:

Open the frontend folder
Right-click index.html
Select Open with Live Server

The application will open automatically in your browser.
